# 功能特性

1. 待做事项列表
   1. 根据事项的配置,按照**提醒值**进行排序
      1. 提醒值=100*(从建立或者上一次打卡过去的时间/预设打卡时间)
   2. 列表可以根据场景(只有工作和生活两种)和标题关键字进行过滤
2. 事项打卡周期可以有多种设置
   1. 预设**有限**次数,打完卡则会自动禁用
   2. **长期**打卡不限次数
   3. 打卡周期可以按:**小时/天/周/月**
3. 打卡
   1. 可以**在列表直接打卡**,也可以进入事项编辑打卡
   2. 打卡的同时也可以写上日志,也可以在没有完成打卡的情况下**只记录日志**
